# Changelog

## v1.0-baseline 2026-06-02
- Базовая версия sort-tier.
- Рекурсивное сканирование каталога.
- Группировка по месяцу создания (YYYY-MM).
- Группировка по расширению внутри месяца.
- Перемещение одиночных файлов в `Singles`.
- Исключение системных файлов по атрибуту FILE_ATTRIBUTE_SYSTEM.
- Явное исключение системных каталогов по умолчанию.
- Dry-run режим по умолчанию.
- Логирование операций в `organizer.log`.
- Обработка коллизий имён с добавлением суффиксов `_1`, `_2`, ...

## 2026-06-03 patch
- Добавлены бытовые форматы графических редакторов и музыкальные форматы.
- Добавлен формат `.iso` в список файлов для обработки.
- Поведение при совпадении файла в целевой папке: сравнение SHA256; при совпадении — автоматическая перезапись.
- Исправление: `.iso` теперь **перемещается** в целевую структуру (как обычные файлы). `exe` и `msi` по-прежнему **копируются** и остаются в исходной папке.
- Восстановление: добавлен скрипт `restore_from_log.py` для отката перемещений по `organizer.log`.

## v1.0.1-baseline 2026-06-09
- Теперь тут есть всё, что задумывалось изначально!
- Сканирует директорию рекурсивно и находит файлы по выбранным категориям
- Группирует файлы по дате (папка вида YYYY-MM-DD)
- Затем распределяет их по расширениям (подпапки: jpg, mp4, pdf и т.д.)
- Структура результата: <целевая_папка>/YYYY-MM-DD/<расширение>/файл
- Обнаруживает **идентичные файлы** (по SHA256) и не дублирует их
- При конфликте имён генерирует уникальные имена (`file_1.jpg`, `file_2.jpg`)
- Исключает **системные файлы** (Windows атрибут)
- Автоматически обходит опасные папки:
  - `C:\Windows`
  - `C:\Program Files` и `C:\Program Files (x86)`
  - `C:\ProgramData`
  - Папки игровых клиентов (Epic Games, GOG, Steam, Origin, Riot Games)
- Поддерживает **dry-run** режим для предварительного просмотра всех операций
- Ведёт **подробный лог** всех операций в файле `sort-tier-<unix_ts>.log`
- Позволяет **полностью откатить изменения** и вернуть файлы обратно в исходные директории
- Создаёт отдельный лог восстановления `sort-tier-back-<unix_ts>.log` в исходной папке
- При копировании восстановление не предлагается
- **Графический UI** (при наличии зависимостей):
  - `InquirerPy` — красивые интерактивные диалоги выбора категорий и режимов
  - `tkinter` — системные диалоги выбора папок
  - `pyfiglet` — ASCII-арт баннер приложения
- **Fallback режим** — автоматический переход на текстовый ввод в консоли, если графические библиотеки недоступны
- **Прогрессбар** (tqdm) для отслеживания процесса обработки (только в консоли, отключён в fallback режиме)
- Расширений стало больше. 
- IMAGE_EXTS = {"jpg","jpeg","png","gif","bmp","webp","tiff","tif","raw"}
- AUDIO_EXTS = {"mp3","m4a","flac","wav","aac","ogg","wma","alac","opus"}
- VIDEO_EXTS = {"mp4","mkv","mov","avi","wmv","flv","webm","m4v", "mp3","mpeg","mpg","3gp"}
- DOC_EXTS = {"pdf","doc","docx","odt","txt","rtf","xls","xlsx","csv","ppt","pptx","md", "epub","fb2","djvu"}
- GRAPHICS_EXTS = {"psd","psb","ai","eps","cdr","xcf","kra","sketch","ps","ora","pdn", "afphoto", "afdesign"}
- INSTALLER_EXTS = {"exe","msi", "msix","appx","deb","rpm","pkg","dmg"}
- ARCHIVE_EXTS = {"zip","rar","7z","tar","gz","bz2","xz","torrent","cab","arj","lzma"}
- ISO_EXTS = {"iso"}
