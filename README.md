# sort-tier

**Краткое описание**  
sort-tier — утилита для Windows, рекурсивно сканирующая каталог и группирующая файлы по месяцу создания (YYYY-MM) и по расширению. Одиночные файлы перемещаются в папку `Singles`. Скрипт исключает системные файлы, поддерживает dry-run, логирование и обработку коллизий имён.
Системные требования: python 3.8 и выше.

## Что изменено в последней версии
- Добавлены форматы графических редакторов и музыкальные форматы.
- Формат `.iso` теперь **перемещается** в целевую структуру (как обычные файлы).
- `exe` и `msi` по-прежнему **копируются** в целевую структуру; оригиналы остаются.
- При совпадении имени и формата в целевой папке сравниваются SHA256; при совпадении — автоматическая перезапись.
